#### Getting Started
1. Clone this repo: ``` git clone https://github.com/sageibra/SpaceX ```
2. Install dependencies: ``` pod install ```
3. Open project

> Requirements
- macOS 10.15 Big Sur or higher
- Xcode 12.x or higher (Swift 5.0)
- CocoaPods [link](https://cocoapods.org/)
